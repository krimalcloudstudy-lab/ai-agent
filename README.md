# 🚨 AI Decision Agent (MCP Based)

## 📌 Overview
This project is an AI-powered decision-making agent using MCP (Model Context Protocol).

## ⚙️ Architecture
- Model: OpenAI
- Context: Real-time Weather API
- Protocol: Flask-based workflow

## 🚀 Features
- Real-time data analysis
- AI-based decisions
- Web interface

## 🌐 Deployment
Deployed using Google Cloud Run

## 🏆 Outcome
Provides instant and intelligent safety decisions.
